# Exercise 02 — Your own dispatch

**Time:** 20 min · **Difficulty:** intermediate

## Goal

Pick one passing candidate from Exercise 01. Dispatch it to the coding agent. Review and merge.

## Steps

1. Open an issue on the demo repo with the dispatch text.
2. `@copilot` mention to assign.
3. While the agent works, draft your reviewer prompt.
4. When the PR opens, run reviewer chat mode.
5. Iterate or merge.

## Acceptance

- A merged PR (or a single, clean iteration cycle).
- Zero Critical findings from reviewer chat mode at merge time.
- ctest green on the merge commit.

## Stretch

Compare your time-to-merge against doing the same task by hand. The first dispatch is usually slower; the third is usually faster.
